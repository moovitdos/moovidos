# Moovitdos - תחבורה ציבורית לכולם

האפליקציה מאפשרת קבלת מידע חכם ומהיר על תחבורה ציבורית, ומותאמת במיוחד לבעלי מכשירים עם סים כשר ומכשירי מקשים.

## איך זה עובד?
קבלת המידע בזמן אמת נעשית באמצעות חיוג אוטומטי מתוך האפליקציה לקווי מידע קוליים (כמו "קל קו"). האפליקציה חוסכת לכם את ההאזנה לתפריטים – היא שולחת באופן אוטומטי את מספר התחנה והקו המבוקשים מיד עם תחילת השיחה, כך שהמידע מתקבל באופן מיידי. 
*(ניתן לשנות את מספר היעד של קו המידע בהגדרות האפליקציה).*

## תכונות מרכזיות
* 🔍 **חיפוש חכם:** איתור מהיר לפי מספר קו, מספר תחנה (מק"ט), שם תחנה, מיקום נוכחי או כתובת.
* ⭐ **מועדפים:** שמירה של תחנות, קווים, מיקומים ואנשי קשר לחיוג מהיר, לגישה נוחה ביומיום.
* 📅 **לוחות זמנים אופליין:** צפייה בלוחות הזמנים המלאים והרשמיים של כל קו – ללא צורך בחיבור לאינטרנט!

## הוראות ניווט למכשירי מקשים
האפליקציה עברה התאמה מיוחדת לשימוש קל ונוח דרך המקשים הפיזיים של המכשיר:
* **מקש כוכבית (*):** חזרה למסך הבית.
* **מקש סולמית (#):** פתיחת מסך ההגדרות.
* **מקש "שמאלי":** ביצוע חיוג (במסכי החיפוש הרגילים).
* **מקש "אישור" (אמצעי):** ביצוע חיוג ישיר (במסך המועדפים).

## עלות ורכישה
